Whiley gets a While Loop!

Thursday, September 23rd, 2010
Finally, in the upcoming release of Whiley, I have added support for both while and for loops — which is about time. Here’s an example: define nat as int where $ >= 0 nat sum([nat] list): r=0 i=0 while i < |list| where r >= 0: r = r + list[i] i = i + 1 return r This code sums a list of natural numbers which, of course, produces a natural number.
