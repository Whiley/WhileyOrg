Install

Whiley is currently written in a combination of Rust and Java. This is for legacy reasons, since the compiler was originally written in Java and is now being migrated to Rust. As such you need to have Java installed on your machine.

The simplest way to install Whiley is through Cargo (see instructions for installing via rustup here). To install Whiley, run cargo as follows:

> cargo install whiley

This will download and compile the latest version of Whiley (note this also requires libssl-dev is installed). You can then test your installation as follows:

> wy

You should then see information about the various commands you can run (e.g. build). See the getting started guide for what to do next.
